# OMEGA AI SaaS Project

## Frontend
Upload `frontend` folder to Netlify.

## Backend
Deploy `backend` folder to Render.

Render settings:
- Build Command: `pip install -r requirements.txt`
- Start Command: `uvicorn main:app --host 0.0.0.0 --port $PORT`

## Features
- Landingpage
- Stripe Pricing Links
- Register/Login
- Dashboard
- Chat API
- FastAPI Backend
